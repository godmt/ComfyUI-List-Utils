# ComfyUI-List-Utils
LIST and BATCH utilities which support: create, convert, get or slice items
## Installation
goto ComfyUI/custom_nodes dir in terminal(cmd)  
git clone https://github.com/godmt/ComfyUI-List-Utils.git  
Restart ComfyUI
